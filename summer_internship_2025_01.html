Codon Degeneracy Explained

Why nature says the same thing in different ways?

theIndianBiologist · May 21, 2025

Codon degeneracy refers to the redundancy of the genetic code — where multiple codons code for the same amino acid. This phenomenon has deep implications in molecular evolution, gene regulation, and translational efficiency.

Redundancy in the Genetic Code

The genetic code contains 64 codons but only 20 standard amino acids. This means some amino acids are encoded by more than one codon. For instance, Leucine is encoded by six different codons.

This redundancy isn’t random. Some codons are used more frequently than others — a phenomenon known as codon usage bias.

Understanding this bias helps explain differences in gene expression between species, and is even exploited in biotechnology.
